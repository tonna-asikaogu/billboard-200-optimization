# Billboard 200 GOAT Optimization Dataset

This repository contains the data needed to reverse-engineer the proprietary weighting system for Billboard's latest "Greatest of All Time Billboard 200 Albums" ranking.

## Files
* **bb200_cache.zip**: A compressed folder containing historical weekly Billboard 200 chart data CSVs.
* **target_rank.csv**: The official, final "True GOAT" rankings published by Billboard in 2015.
